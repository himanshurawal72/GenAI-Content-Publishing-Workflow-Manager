# GenAI Content Publishing Workflow Manager

This project is an agentic AI-based system designed to automate research, content generation, quality review, and professional PDF publishing.

It uses a structured, multi-agent workflow to ensure content accuracy, quality control, and human-in-the-loop feedback before final publication.

---

## Project Objectives

- Automate research using AI-assisted web search
- Generate structured and professional content
- Perform internal quality review before final output
- Allow human feedback and iterative improvement
- Export finalized content as a professional PDF

---

## Tools & Technologies Used

- **Python** – Core programming language
- **LangGraph** – Agentic workflow orchestration and state management
- **Streamlit** – Web-based user interface for interaction
- **Groq LLM (LLaMA-based model)** – Content generation and reasoning
- **Tavily API** – Real-time web research and source gathering
- **ReportLab** – Professional PDF generation
